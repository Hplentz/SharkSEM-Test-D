auto_functions.run_auto_lens_alignment

Runs a routine that automatically adjusts the lens alignment in the active view.

Syntax

run_auto_lens_alignment()
run_auto_lens_alignment(settings)

Parameters

settings RunAutoLensAlignmentSettings Settings for the automatic lens alignment adjustment routine.

Remarks

This routine only works with the electron beam, which must be selected as the imaging device in the active view.
There is only one algorithm variant for the routine: Standard.

Examples


    # Run the standard auto lens alignment routine
    microscope.auto_functions.run_auto_lens_alignment()
    
    # Run the standard auto lens alignment routine, providing additional parameters
    settings = RunAutoLensAlignmentSettings(dwell_time=1e-6, reduced_area=Rectangle(0.1, 0.1, 0.3, 0.3))
    microscope.auto_functions.run_auto_lens_alignment(settings)
