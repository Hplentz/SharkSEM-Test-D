specimen.compustage

Enables control and provides the status of the compustage device.

Classes

neutral_mill_position Enables control and provides the status of the compustage neutral mill position.

Properties

current_position [Read only] Gets the current compustage position.
is_homed [Read only] Returns True if the compustage is homed.
is_installed [Read only] Tells whether the compustage is installed on the system. This property does not indicate whether the compustage rod is loaded or inserted.
is_linked [Read only] Returns True if the sample Z is linked to the free working distance.

Methods

absolute_move() Moves the compustage to the specified position.
flip() Flips the compustage by 180 degrees.
home() Homes all axes of the compustage.
insert() Inserts the compustage.
link() Links the sample Z to the free working distance.
relative_move() Moves the compustage by the specified delta position.
retract() Retracts the compustage.
