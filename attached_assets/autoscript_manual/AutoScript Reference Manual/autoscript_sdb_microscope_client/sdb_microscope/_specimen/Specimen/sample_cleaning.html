specimen.sample_cleaning

Enables control and provides the status of the sample cleaning.

Properties

is_installed [Read only] Returns True if the plasma cleaning device is present on the microscope.

Methods

run() Executes the sample cleaning procedure synchronously, i.e. waits for the sample cleaning to end.

Remarks

Chamber needs to be pumped into the high vacuum before sample cleaning is started.

Examples


        # Performing sample cleaning with specified duration, if installed.
        if microscope.specimen.sample_cleaning.is_installed:
            microscope.specimen.sample_cleaning.run(30)
