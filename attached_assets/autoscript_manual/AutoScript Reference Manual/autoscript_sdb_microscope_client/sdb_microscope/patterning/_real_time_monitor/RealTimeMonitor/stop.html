patterning.real_time_monitor.stop

Stops the real time monitoring process.

Syntax

stop()

Remarks

  • Calling this function does not clear RTM data and positions. They can still be retrieved with patterning.real_time_monitor.get_data() and patterning.real_time_monitor.get_positions() even after RTM has been stopped.
  • Each patterning.real_time_monitor.stop() call should be preceded by patterning.real_time_monitor.start() call.
