beams.electron_beam.beam_shift.value

Gets or sets the value.

Data type

Point

Remarks

Beam shift values are specified in meters.

Examples


        # Set new X and Y values of beam shift
        microscope.beams.electron_beam.beam_shift.value = Point(1e-6, 1e-6)

        # WRONG: Writing to x or y property directly has no effect
        microscope.beams.electron_beam.beam_shift.value.x = 1e-6

        # Setting beam shift only in one axis requires a workaround
        current_beam_shift_y = microscope.beams.electron_beam.beam_shift.value.y
        microscope.beams.electron_beam.beam_shift.value = Point(1e-6, current_beam_shift_y)
