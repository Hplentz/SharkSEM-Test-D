beams.electron_beam.scanning.mode.set_reduced_area

Activates the reduced area scanning mode.

Syntax

set_reduced_area()
set_reduced_area(left, top, width, height)

Parameters

left float Left side of the rectangular area. The coordinate is in the range [0, 1].
top float Top side of the rectangular area. The coordinate is in the range [0, 1].
width float Width of the rectangular area. The valid range is [0, 1], where 1 means the full width of the view.
height float Height of the rectangular area. The valid range is [0, 1], where 1 means the full height of the view.

Remarks

Once this scanning mode is active, it has an impact on the method start_acquisition(). Acquisition is performed only in the selected area.

If rectangle definition is passed, it also sets the position and size of the reduced area. If no parameters are passed, the last used reduced area is set.

If it is desired to grab a reduced area of a frame using method grab_frame(), use structure GrabFrameSettings as a parameter of the method grab_frame(). For more information, read the documentation of grab_frame().

Examples


        # The reduced area will start in the top left quarter of the view
        # and will reach to the right bottom corner of the view
        microscope.beams.electron_beam.scanning.mode.set_reduced_area(0.25, 0.25, 0.75, 0.75)
        microscope.imaging.start_acquisition()
