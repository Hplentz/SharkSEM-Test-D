beams.ion_beam.scanning.mode.set_line

Activates the line scanning mode.

Syntax

set_line()
set_line(y)

Parameters

y float The position of the line in a view. Valid values are in the range [0, 1], where 0 represents top of the view and 1 represents bottom of the view.

Remarks

Once this scanning mode is active, it has an impact on the method start_acquisition(). Acquisition is performed along the selected line.

If parameter y is set, it also sets position of the line in an active view. If no parameter is passed, the last line position is used.

Examples


        # Activate line-scanning mode and set position of the line to 0.7
        microscope.beams.ion_beam.scanning.mode.set_line(0.7)
        
        # Activate full-frame scanning mode
        microscope.beams.ion_beam.scanning.mode.set_full_frame()
        
        # Activate line scanning mode and set position to the last used (0.7)
        microscope.beams.ion_beam.scanning.mode.set_line()
