vacuum.pump

Pumps the specimen chamber into vacuum or changes vacuum conditions according to the specified settings.

Syntax

pump()
pump(settings)

Parameters

settings VacuumSettings A structure specifying vacuum mode, pressure and gas.

Remarks

  • Calling pump() when the chamber is already pumped does nothing.
  • Calling pump(settings) with settings different from current conditions will bring the vacuum to the desired state.
  • If the vacuum system is in the "venting" transition state at the time the pump() function is called, the function will interrupt the transition in progress and begin pumping the chamber.
  • Note that the change in chamber pressure is executed asynchronously. The pump() function may return before the specified chamber pressure is reached.

Examples


    settings = VacuumSettings(mode=VacuumMode.LOW_VACUUM, gas=VacuumGasType.WATER, pressure=120)
    microscope.vacuum.pump(settings)
