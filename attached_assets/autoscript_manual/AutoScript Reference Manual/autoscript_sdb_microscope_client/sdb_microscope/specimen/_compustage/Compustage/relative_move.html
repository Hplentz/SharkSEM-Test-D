specimen.compustage.relative_move

Moves the compustage by the specified delta position.

Syntax

relative_move(position_delta)
relative_move(position_delta, settings)

Parameters

position_delta CompustagePosition The position by which the compustage should move.
settings MoveSettings Additional move settings.

Examples


        # Move the compustage by -0.2mm in X axis
        delta_position = CompustagePosition(x=-0.0002)
        microscope.specimen.compustage.relative_move(delta_position)
