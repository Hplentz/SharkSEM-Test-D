specimen.manipulator.absolute_move

Moves the manipulator to the specified position.

Syntax

absolute_move(target_position)

Parameters

target_position ManipulatorPosition The absolute position to which the manipulator should move.

Remarks

Whether it is possible to rotate the manipulator needle depends on the installed version of the manipulator. If the R axis is specified for movement, but the manipulator does not support rotation, an exception is raised

Examples


        position = ManipulatorPosition(x=0.0001, y=0.0001)
        microscope.specimen.manipulator.absolute_move(position)
