specimen.manipulator.insert

Inserts the manipulator either to a default or specified position.

Syntax

insert()
insert(insert_position)

Parameters

insert_position ManipulatorPosition The position to which the manipulator should be inserted.

Remarks

  • The parameterless function inserts the manipulator into a position from which it was previously retracted, or Eucentric when the manipulator is being inserted for the first time.
  • If the insertion position is specified in Stage coordinate system, the manipulator will reach the position after the needle has been inserted.
  • The meaning of insert_position parameter is different if the position is given in Raw coordinate system. In this case the retracted manipulator is first moved to this position and then inserted. It is therefore a position from which the manipulator should be inserted.

Examples


    # Insert the manipulator needle into the Park position
    park_position = microscope.specimen.manipulator.get_saved_position(ManipulatorSavedPosition.PARK)
    microscope.specimen.manipulator.insert(park_position)
