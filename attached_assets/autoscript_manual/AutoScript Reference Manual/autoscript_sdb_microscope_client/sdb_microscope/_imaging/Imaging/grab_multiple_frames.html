imaging.grab_multiple_frames

Scans new frames in all compatible views and provides the corresponding images.

Syntax

grab_multiple_frames()
grab_multiple_frames(settings)

Parameters

settings GrabFrameSettings The custom scan settings to be used instead of default values.

Return value

List[AdornedImage] The list of microscope images, each corresponding to a single compatible view.

Remarks

  • The function initiates scanning of new images and waits until they are completely scanned.
  • The scan is performed in all views that are compatible with the active view. Compatible views are those which have been assigned a) the same imaging device and b) a compatible detector.
  • Custom scan settings can be provided by the settings parameter. If not specified, the actual scan settings are determined as follows:
    • resolution, dwell_time and reduced_area will default to the current settings in the active view.
    • Other settings will fallback to predefined constant default values:
      • bit_depth will be 8.
      • line_integration will be 1.
      • scan_interlacing will be 1.
      • frame_integration will be 1.
      • drift_correction will be False.
  • The grab_multiple_frames() function can perform image acquisition only in standard resolutions, that is resolutions from the ScanningResolution enumeration. Maximum resolution is therefore 6144x4096. An error is returned for other resolutions.
  • Parameter preview_resolution specified in settings is ignored by grab_multiple_frames().

Examples


    settings = GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024, frame_integration=4)
    images = microscope.imaging.grab_multiple_frames(settings)

    print(f"Acquired {len(images)} images.")
    

        # Acquire multiple images by integrating 8 consecutive frames in all compatible views and correcting them for drift
        settings = GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024, dwell_time=100e-9,
                                     frame_integration=8, drift_correction=True)
        images = microscope.imaging.grab_multiple_frames(settings)
