imaging.get_image

Retrieves a microscope image currently present in the active view.

Syntax

get_image()

Return value

AdornedImage The microscope image retrieved from the active view.

Remarks

The method does not initiate the scanning of a new image frame. It inspects the imaging view and returns the image stored there from previous calls to grab_frame().
If you want to acquire a new image with the active imaging device, call grab_frame() instead.
