imaging.grab_frame

Scans a new frame in the active view and provides the corresponding image.

Syntax

grab_frame()
grab_frame(settings)

Parameters

settings GrabFrameSettings The custom scan settings to be used instead of default values.

Return value

AdornedImage The microscope image corresponding to the acquired frame.

Remarks

  • The function initiates scanning of a new frame and waits until it is completely scanned.
  • Custom scan settings can be provided by the settings parameter. If not specified, the actual scan settings are determined as follows:
    • resolution, dwell_time and reduced_area will default to the current settings in the active view.
    • Other settings will fallback to predefined constant default values:
      • bit_depth will be 8.
      • line_integration will be 1.
      • scan_interlacing will be 1.
      • frame_integration will be 1.
      • drift_correction will be False.
  • Maximum resolution accepted by grab_frame() is 6144x6144. For higher resolutions use imaging.grab_frame_to_disk().
  • It is recommended to use one of the standard resolutions for common imaging tasks, that is a resolution from the ScanningResolution enumeration. If a non-standard resolution is specified in settings, the function will use a less effective method of imaging and some of the scan parameters will not be applicable. See User guide: Imaging and User guide: Large image acquisition for more information.
  • Parameter preview_resolution specified in settings is ignored by the grab_frame() function.

Examples


        from matplotlib import pyplot as plot
        
        # Take one image with the specified resolution and dwell time and display it on a pop-up window
        settings = GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024, dwell_time=500e-9)
        image = microscope.imaging.grab_frame(settings)
        
        plot.imshow(image.data, cmap='gray')
        plot.show()
    

        from matplotlib import pyplot as plot

        # Take one image in a rectangular sub-region of the full frame, so called reduced area, and display it on a pop-up window
        settings = GrabFrameSettings(reduced_area=Rectangle(0.60, 0.60, 0.30, 0.30))
        image = microscope.imaging.grab_frame(settings)
        
        plot.imshow(image.data, cmap='gray')
        plot.show()
    

        # Acquire an image by integrating 8 consecutive frames and correcting them for drift
        settings = GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024, dwell_time=100e-9,
                                     frame_integration=8, drift_correction=True)
        image = microscope.imaging.grab_frame(settings)
    

        # Acquire an image with a non-standard resolution of 800x200
        settings = GrabFrameSettings(resolution="800x200")
        image = microscope.imaging.grab_frame(settings)
