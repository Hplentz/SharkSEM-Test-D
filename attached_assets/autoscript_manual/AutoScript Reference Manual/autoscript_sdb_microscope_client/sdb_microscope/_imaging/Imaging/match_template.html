imaging.match_template

Tries to locate a feature from the given template image in the given searched image.

Syntax

match_template(image, template_image)

Parameters

image AdornedImage An image to be searched for the feature from the given template image.
template_image AdornedImage An image containing the template of the feature to be located.

Return value

ImageMatch The possible location of the feature, accompanied by a confidence score in a range of [0, 1].

Remarks

  • The function uses the HOG (histogram of gradients) template matching algorithm to find the feature. The matching algorithm cannot be configured as with the locate_feature() function from the AutoScript Vision Toolkit.
  • The algorithm is size invariant - it requires that the template image has the same pixel size as the area containing the feature in the target image.
  • Only one possible position of the feature is returned as output, accompanied with a confidence score in a range of [0, 1] corresponding to 0% - 100%.
