detector.custom_settings

Enables control and provides the status of the additional detectors settings.

Classes

converter_voltage Enables control and provides the status of the active detector converter voltage.
grid_voltage Enables control and provides the status of the active detector grid voltage.
md_shutter Represents the Mirror Detector Shutter.
mirror_voltage Enables control and provides the status of the active detector mirror voltage.
scintillator_voltage Enables control and provides the status of the active detector scintillator voltage.
suction_tube_voltage Enables control and provides the status of the active detector suction tube voltage.

Methods

get_segment_polarity() Gets the polarity of a segment.
list_all_segments() Returns all segments available on the active detector.
reset_segments() Resets all eligible active detector segments in custom mode.
set_segment_polarity() Sets the polarity of a segment.
