Microscope API

The Python package autoscript_sdb_microscope_client enables the control of microscope systems from Thermo Fisher Scientific SEM (scanning electron microscope) and DB (dual beam) families.

The most important class in this package is SdbMicroscopeClient, which after instantiation and connection to an AutoScript server provides a wide range of functions through its members (child objects, members and functions) organized in a static tree.

The package also contains other classes. Data structures make parameters and return values of static tree methods more comprehensive, while enumerations can save some time and help avoid typos when used instead of string or integer values. Dynamic objects are objects whose lifetime is expected to be shorter than the lifetime of SdbMicroscopeClient and which often have to be created explicitly during script execution. Patterns can serve as a good example of dynamic objects.
