AutoFunctionMethod

An enumeration of all auto function methods.

Values

MAX_CONTRAST "MaxContrast" Method to prioritize contrast maximization.
ONG_ET_AL "OngEtAl" Method based on research of Ong et al.
STANDARD "Standard" Standard method for wide range of use cases.
VOLUMESCOPE "Volumescope" Method based on Volumescope workflow specialized for biological samples.

Remarks

Method to be used for particular auto function run can be configured via Method member of a setting structure passed to the auto function method.
