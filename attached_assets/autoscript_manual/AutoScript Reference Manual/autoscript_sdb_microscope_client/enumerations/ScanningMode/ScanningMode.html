ScanningMode

Specifies the available modes of beam scanning.

Values

CROSSOVER 7 The image of the electron source tip is acquired instead of the sample surface.
EXTERNAL 5 The external control of the scanning system is used, such as beam control from an EDX X-ray system.
FULL_FRAME 1 The whole frame is acquired.
LINE 2 Only one line of the frame is acquired.
OTHER 6 Unknown scan mode.
REDUCED_AREA 4 The specified rectangular area is acquired.
SPOT 3 Only one pixel of the frame is acquired.

Remarks

The enumeration can be used to explain the value of the properties electron-beam.scanning.mode.value and ion-beam.scanning.mode.value.
