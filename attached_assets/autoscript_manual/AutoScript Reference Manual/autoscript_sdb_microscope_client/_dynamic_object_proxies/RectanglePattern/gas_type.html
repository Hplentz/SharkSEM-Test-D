RectanglePattern.gas_type

The name of the gas that must be used to pattern this shape or empty string if no gas is to be used.

Data type

str

Examples


rectangle_pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)

# Assign a single gas with a MultiChem
rectangle_pattern.gas_type = "A-Gas"
print(rectangle_pattern.gas_type)
# Gas duty cycle 10
duty_cycle = [10]
rectangle_pattern.gas_flow = duty_cycle
print(rectangle_pattern.gas_flow)

# Assign a mixed gas with a MultiChem
rectangle_pattern.gas_type = "A-Gas+B-Gas"
print(rectangle_pattern.gas_type)
# Gas duty cycle 10, 90
duty_cycle = [10, 90]
rectangle_pattern.gas_flow = duty_cycle
print(rectangle_pattern.gas_flow)
