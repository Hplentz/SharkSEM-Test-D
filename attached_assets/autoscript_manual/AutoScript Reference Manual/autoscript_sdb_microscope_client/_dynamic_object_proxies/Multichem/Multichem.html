Multichem

Represents the MultiChem gas injection device.

Methods

close() Closes the gas flow in MultiChem.
get_temperature() Determines the temperature of the specified gas in MultiChem. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().
insert() Inserts the MultiChem needle into one of the predefined positions.
list_all_gases() Lists all gases in MultiChem.
open() Opens the gas flow in MultiChem.
retract() Retracts the MultiChem needle if inserted.
turn_heater_off() Turns the MultiChem heater off for the specified gas.
turn_heater_on() Turns the MultiChem heater on for the specified gas.
