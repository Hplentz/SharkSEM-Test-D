Multichem.insert

Inserts the MultiChem needle into one of the predefined positions.

Syntax

insert()
insert(position)

Parameters

position str The name of a MultiChem insert position.

Remarks

  • Enumeration MultiChemInsertPosition lists standard insert positions. Note that not all positions may be available on a particular system.
  • The parameterless function inserts the needle into a default position. On systems equipped with a bulk stage the default position is determined by the current stage tilt (MultiChemInsertPosition.ELECTRON_DEFAULT for tilt angle 0°, MultiChemInsertPosition.ION_DEFAULT for tilt angle 52°).
    When Compustage is present and active, the default position is determined differently - please refer to XT User Manual for details.
  • If the MultiChem needle is already inserted in any position, the function returns immediately without any action being taken. To move the inserted needle from one position to another, you must explicitly call retract().

Examples


        print("Linking bulk stage Z coordinate to working distance...")
        microscope.specimen.stage.link()
        multichem = microscope.gas.get_multichem()
        print("Inserting MultiChem...")
        multichem.insert()
        print("Retracting MultiChem...")
        multichem.retract()
    

        multichem = microscope.gas.get_multichem()
        multichem.insert(MultiChemInsertPosition.COMPUSTAGE0_DEFAULT)
