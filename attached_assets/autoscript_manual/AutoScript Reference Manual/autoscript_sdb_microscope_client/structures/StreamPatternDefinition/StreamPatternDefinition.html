StreamPatternDefinition

Contains the definition of a stream pattern.

Items

raw_points bytes Patterning points in a raw form.
repeat_count int The number of patterning cycles.
bit_depth int Bit depth of the point coordinates. It can be either 12 or 16.
points Numpy.ndarray Patterning points in a form of 2D numpy.ndarray.

Methods

load() Loads the stream pattern definition from a stream file.

Remarks

Stream pattern definition can be initialized in the following ways:

  • Loaded from a stream file: use static load() method.
  • Created directly in the script: at least points and repeat_count properties need to be set, while bit_detph defaults to 16.

The above mentioned ways can also be combined: you can load the definition from a stream file and then adjust it in the script.

Patterning points can be manipulated in a form of 2D numpy.array via points property. In this array, each point is represented by a 4-tuple [x, y, dwell_time, flags], with the values having the following semantics:

  • x is a horizontal coordinate of the point in a range of [0, 2bit_depth], raising from left to right.
  • y is a vertical coordinate of the point in a range of [0, 2bit_depth], raising from top to bottom.
  • dwell_time is dwell time setting in seconds, which has to be a multiple of 25ns for bit_depth = 16 and a multiple of 100ns for bit_depth = 12.
  • flags specify additional point properties: use 0 for no flags, 1 to blank the point.

Please note that X and Y coordinates are subject to offset defined by stream pattern center. Point in the center of available ranges (e.g., [32768, 32768] for bit_depth = 16) maps to the center of the stream pattern.

Examples

The following example shows how to create stream pattern definition directly in the script:


        import numpy
        
        spd = StreamPatternDefinition()
        spd.points = numpy.array([[1000, 11922, 1e-06, 0],  # X = 1000, Y = 11922, dwell_time = 1us, flags = 0 (no blanking)
                                  [32767, 11922, 1e-06, 0],
                                  [64535, 11922, 1e-06, 0],
                                  [64535, 53613, 1e-06, 0],
                                  [32767, 53613, 1e-06, 0],
                                  [1000, 53613, 1e-06, 0]])
        spd.repeat_count = 1000

The following example shows how to load stream pattern definition from a stream file:


        stream_file_path = "c:\\ProgramData\\Thermo Scientific AutoScript\\UserGuide\\Resources\\grid_pattern_definition.str"
        stream_pattern_definition = StreamPatternDefinition.load(stream_file_path)
