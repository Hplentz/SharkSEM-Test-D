AdornedImage

Contains microscope image data along with metadata describing the state of the microscope under which the image was acquired.

Items

width [Read only] int The width of the image.
height [Read only] int The height of the image.
bit_depth [Read only] int The bit depth of the image data.
metadata [Optional] AdornedImageMetadata Metadata containing information about system state at the time this image was acquired.
data [Read only] Numpy.ndarray Image data arranged either in 2D (for grayscale images) or 3D (for RGB images) numpy array.
encoding [Read only] int Encoding of the image. To explain this property use enumeration ImageDataEncoding.
checksum [Read only] int Checksum of the raw-data.
thumbnail [Read only] AdornedImage Thumbnail of the original image.

Methods

load() Loads an image from a TIFF file.
save() Saves the image at the specified path as a TIFF file.

Remarks

The new AdornedImage can be constructed either empty or from the given data if passed. The data has to be in a format of a numpy.ndarray.
