RunAutoStigmatorSettings

Contains settings for the automatic stigmator adjustment routine.

Items

method [Optional] str The name of the the auto stigmator algorithm variant.
dwell_time [Optional] float Dwell time to be used during the routine.
horizontal_field_width [Optional] float Horizontal field width to be used during the routine.
line_integration [Optional] int Line integration to be used during the routine.
number_of_frames [Optional] int The number of frames to be used during the routine.
reduced_area [Optional] Rectangle Reduced area to be used during the routine. Point [0,0] corresponds to the upper left corner of the image, point [1, 1] to the lower right corner.
resolution [Optional] str Scanning resolution to be used during the routine.
stigmation_step [Optional] float Initial stigmation step to be used by the routine while searching for the optimal stigmator setting.
display_point [Optional] Point A point in imaging view from which the reduced area position will be calculated. Point [0,0] corresponds to the upper left corner of the view, point [1, 1] to the lower right corner.
