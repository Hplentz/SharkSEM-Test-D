AutoScript Toolkit

AutoScript Toolkit is a Python package that provides data processing capabilities in selected areas of electron microscopy. This package is a part of the AutoScript software suite. For more information, see other parts of this reference manual, in particular the Microscope API package.

The toolkit contains three modules. Image-template matchers can be found in the module Template matchers. Help for computer vision tasks is provided through Vision packages. Another package, drift_correction, applies computer vision to a specific area of the sample drift correction.
