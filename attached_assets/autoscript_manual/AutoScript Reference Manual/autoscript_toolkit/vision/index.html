Computer vision

This module provides aid for computer vision tasks.

Classes

FeatureLocation This class contains various information about a feature located in an image.

Methods

cut_image() This method cuts out a part of the given image according to the given target area definition.
locate_feature() This method locates a feature according to the given template in the given image.
plot_match() This method presents a searched image and a feature located in it.

Remarks

This module uses the template_matchers module for image template matching purposes.
