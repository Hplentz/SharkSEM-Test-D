FeatureLocation

This class contains various information about a feature located in an image. It is a result type of locate_feature() method.

Methods

print_all_information() This method prints all available information in a well-formatted, human-readable manner.

Properties

confidence Level of confidence with which the feature was located in range between [0, 1] corresponding to 0% - 100%.
center_in_pixels Coordinates of the center of the feature in the original image, in pixels.
center_in_meters Coordinates of the center of the feature in the original image, in meters. For more information on coordinate system, see the notes below.
shift_in_pixels Shift of feature from its original position, in pixels. See the notes below for more information.
shift_in_meters Shift of feature from its original position, in meters. See the notes below for more information.

Remarks

Coordinate system used for members referring to meters (center_in_meters, shift_in_meters) is the same as the coordinate system used for patterns, with origin [0, 0] in the center of the view, X axis coordinates growing from left to right and Y axis coordinates growing from bottom to top.

Properties referring to shift (shift_in_pixels, shift_in_meters) are only available when original position of the feature is provided to the locate_feature() method via its original_feature_center parameter.
