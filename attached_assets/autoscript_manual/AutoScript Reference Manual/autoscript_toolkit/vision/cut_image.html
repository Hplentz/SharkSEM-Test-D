cut_image

Cuts out a portion of an image according to the specified target area definition.

Syntax

cut_image(image, [top_left, bottom_right, center, size, relative_center, relative_size])

Parameters

image AdornedImage or numpy.ndarray Image to cut out a part from.
top_left list of size of 2 Top left point of the cut area, in pixels.
bottom_right list of size of 2 Bottom right point of the cut area, in pixels.
center list of size of 2 Center of the cut area, in pixels.
size list of size of 2 Size of the cut area, in pixels.
relative_center list of size of 2 Center of the cut area, in relative coordinates in the range [0, 1].
relative_size list of size of 2 Size of the cut area, in relative coordinates in the range [0, 1].

Return value

AdornedImage or numpy.ndarray Cut taken from the image in the same format as the input image.

Remarks

Please note that only the following combinations of parameters are valid: top left and bottom right, center and size, or relative center and relative size.

The method can accept both AdornedImage and numpy.ndarray for image parameter. The cut is then provided in the same form.
