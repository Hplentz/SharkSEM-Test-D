DcPatterningEngine.define_fiducial

This method defines fiducial according to the given center point and size.

Syntax

define_fiducial(image, center, size)

Parameters

image AdornedImage Image to be used for fiducial definition.
center float Center point of the fiducial, in meters. For more information on coordinate system, see the notes below.
size float Size of the fiducial, in meters. For more information on coordinate system, see the notes below.

Remarks

Coordinate system used for parameters referring to meters (center, size) is the same as the coordinate system used for patterns, with origin [0, 0] in the center of the view, X axis coordinates growing from left to right and Y axis coordinates growing from bottom to top.
