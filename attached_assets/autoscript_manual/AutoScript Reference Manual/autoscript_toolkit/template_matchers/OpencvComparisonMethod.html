OpencvComparisonMethod

This enumeration lists possible comparison methods for the OpenCV template matching algorithm. It allows convenient setup of the OpencvTemplateMatcher.

Values

CCOEFF 0 TM_CCOEFF method.
CCOEFF_NORMED 1 TM_CCOEFF_NORMED method.
CCORR 2 TM_CCORR method.
CCORR_NORMED 3 TM_CCORR_NORMED method.
SQDIFF 4 TM_SQDIFF method.
SQDIFF_NORMED 5 TM_SQDIFF_NORMED method.

Remarks

Please refer to OpenCV reference manual for more information on the methods.
