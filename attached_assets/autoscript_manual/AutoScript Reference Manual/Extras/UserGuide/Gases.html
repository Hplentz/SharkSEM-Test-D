User guide: Gases

This chapter shows how to control gas injection systems including GIS and MultiChem.

Before proceeding with the code snippets listed below, initialize AutoScript with the following code. Do not forget to set the server address or host name (parameters of the connect() function) according to your environment. 


    from autoscript_sdb_microscope_client import SdbMicroscopeClient
    from autoscript_sdb_microscope_client.enumerations import *
    from autoscript_sdb_microscope_client.structures import *
    
    microscope = SdbMicroscopeClient()
    microscope.connect("localhost")

Get list of available GIS (gas injection system) ports and inspect it.


    ports = microscope.gas.list_all_gis_ports()
    print(ports)

Insert and retract platinum deposition GIS needle:


    # First, link stage Z coordinate to working distance to enable needle insertion
    microscope.specimen.stage.link()

    platinum_gis = microscope.gas.get_gis_port("Pt dep")
    platinum_gis.insert()
    platinum_gis.retract()

Insert and retract MultiChem needle:


    # First, link stage Z coordinate to working distance to enable needle insertion
    microscope.specimen.stage.link()
    
    multichem = microscope.gas.get_multichem()
    multichem.insert()
    multichem.retract()
