User guide: Auto functions

This chapter shows how to run auto functions - automated routines for optimizing microscope optics and detector parameters for the best possible quality of the acquired images.

Before proceeding with the code snippets listed below, initialize AutoScript with the following code. Do not forget to set the server address or host name (parameters of the connect() function) according to your environment. 


    from autoscript_sdb_microscope_client import SdbMicroscopeClient
    from autoscript_sdb_microscope_client.enumerations import *
    from autoscript_sdb_microscope_client.structures import *
    
    microscope = SdbMicroscopeClient()
    microscope.connect("localhost")

Setup imaging environment:


    microscope.imaging.set_active_view(1)
    microscope.imaging.set_active_device(ImagingDevice.ELECTRON_BEAM)

Run auto contrast/brightness routine in active view using default method:


    microscope.auto_functions.run_auto_cb()

Run auto contrast/brightness using specialized MaxContrast method:


    settings = RunAutoCbSettings(method="MaxContrast")
    microscope.auto_functions.run_auto_cb(settings)

Run auto contrast/brightness using specialized MaxContrast method and set up specific settings:


    settings = RunAutoCbSettings(method="MaxContrast", resolution="192x128", calibrate_detector=True)
    microscope.auto_functions.run_auto_cb(settings)

Run auto stigmator using the OngEtAl method:


    microscope.auto_functions.run_auto_stigmator(RunAutoStigmatorSettings(method="OngEtAl"))

Run auto stigmator centering using the Volumescope method:


    microscope.auto_functions.run_auto_stigmator_centering(RunAutoStigmatorCenteringSettings(method="Volumescope"))

Other auto functions can be run in a similar way. Please take a look at the reference manual for more information on available auto functions and their settings.
