User guide: Client startup

This chapter shows how to start an AutoScript client.

Prior to proceeding with any code snippets below, import AutoScript packages using the following code:


from autoscript_sdb_microscope_client import SdbMicroscopeClient
from autoscript_sdb_microscope_client.enumerations import *
from autoscript_sdb_microscope_client.structures import *

First, create microscope client object:


microscope = SdbMicroscopeClient()

Connect the client to the AutoScript server running on the Microscope computer with a standard IP address 192.168.0.1:


microscope.connect("192.168.0.1")

Connect the client to the server at this computer:


microscope.connect("localhost")

By default, client tries to connect to the standard address 192.168.0.1:


microscope.connect()

Connect the client to the server with certain host name, e.g. MPC-TENEO-A2:


microscope.connect("MPC-TENEO-A2")

Disconnect the client:


microscope.disconnect()
