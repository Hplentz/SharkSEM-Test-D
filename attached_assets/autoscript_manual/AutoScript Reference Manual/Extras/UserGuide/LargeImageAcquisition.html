User guide: Large image acquisition

This chapter shows you how to acquire images with non-standard resolutions, that is resolutions not listed in the ScanningResolution enumeration. This method of image acquisition supports any resolution up to 65536x65536. Please read the first paragraph in User guide: Imaging to also learn about disadvantages of large image acquisition technique.

The functionality is accessible via two functions: imaging.grab_frame() and imaging.grab_frame_to_disk().
For resolutions above 6144x6144 the function imaging.grab_frame(), which treats images as in-memory objects, is no longer available due to data size.

Acquisition with the function imaging.grab_frame_to_disk() results in a raw or tiff file on disk. The format is determined by the file_format parameter. The filename extension has no influence on the file format. The function returns a down-scaled ("preview") image instead of the full-scale image. Resolution of the preview image can be specified optionally. If it is not specified, it will be calculated automatically.

Before proceeding with the code snippets below, initialize AutoScript client with the following code, and specify a correct IP address of the AutoScript server. 


    from autoscript_sdb_microscope_client import SdbMicroscopeClient
    from autoscript_sdb_microscope_client.enumerations import *
    from autoscript_sdb_microscope_client.structures import *
    
    microscope = SdbMicroscopeClient()
    microscope.connect("localhost")

Set up imaging environment.
The 1st (upper left) view will become active and will be configured with the electron beam imaging device. 


    microscope.imaging.set_active_view(1)
    microscope.imaging.set_active_device(ImagingDevice.ELECTRON_BEAM)

Acquire an 8k image and save it to disk in the RAW format:


    settings = GrabFrameSettings("8192x8192")
    microscope.imaging.grab_frame_to_disk("C:\\image.raw", ImageFileFormat.RAW, settings)

Acquire an image with resolution 20000x20000, save it to disk in the RAW format and display its preview on a pop-up window:


    from matplotlib import pyplot as plot
    
    settings = GrabFrameSettings("20000x20000", preview_resolution="1000x1000")
    preview_image = microscope.imaging.grab_frame_to_disk("C:\\image.raw", ImageFileFormat.RAW, settings)
    
    plot.imshow(preview_image.data, cmap='gray')
    plot.show()

Acquire a 16k image and save it to disk in the TIFF format, including metadata:


    settings = GrabFrameSettings("16384x16384")
    microscope.imaging.grab_frame_to_disk("C:\\image.tif", ImageFileFormat.TIFF, settings)

Acquire 16k image with 16-bit image depth and line integration of 4:


    settings = GrabFrameSettings("16384x16384", bit_depth=16, line_integration=4)
    microscope.imaging.grab_frame_to_disk("C:\\image.tif", ImageFileFormat.TIFF, settings)

Acquire an image at non-standard resolution 2000x1000 to memory:


    settings = GrabFrameSettings("2000x1000")
    image = microscope.imaging.grab_frame(settings)
    
    print(image.width, image.height)
