User guide: Package import

This chapter shows how to import AutoScript packages into a Python script.

Import SDB microscope client from SDB microscope package:


from autoscript_sdb_microscope_client import SdbMicroscopeClient

Import enumerations and structures from SDB microscope package:


from autoscript_sdb_microscope_client.enumerations import *
from autoscript_sdb_microscope_client.structures import *
