Python Distribution

AutoScript contains Python 3 distribution from Enthought. In addition to the 64-bit Python interpreter the distribution includes a selection of pre-installed packages for scientific computing, image and data processing.

When the AutoScript product is installed, an initial Python environment with the name "AutoScript" is created. You can manage packages within this environment as well as create or remove new environments using the standard tooling provided by Enthought - the Enthought Deployment Manager command line tool - "Edm".

Examples:

To list all installed packages in the AutoScript environment, type in the Windows command line prompt: 
edm list -e AutoScript
To download the latest version of the "arrow" package from Enthought Deployment Server and install it to the AutoScript environment: 
edm install arrow -e AutoScript
Note that the computer must be connected to the Internet for the second example to work.

For more information about using the Edm tool, see the Enthought website online at https://docs.enthought.com/edm.
