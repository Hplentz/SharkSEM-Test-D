Depending on microscope type the function turn off the Ga filament heating or turn off the Xe plasma source and close the gas valve.

◆ FibHVBeamOn()

static void SharkSEM.Commands.FibHVBeamOn ( this Connection  Conn)
inlinestatic

Turn on the FIB beam.

Depending on microscope type the function turn on the Ga filament heating or open the gas valve, start regulation and turn on the Xe plasma source.

If the main accelerating voltage is off, it is turned on automatically, prior to beam on procedure.

It is recommended to poll the beam status using FibHVGetBeam() till its result is <=1, meaning that the procedure has been completed.

Timing
The execution time may vary. If waiting for F flag is not requested, it is executed immediately. If waiting F flag is requested, it will wait until procedure is done. Wait F flag is set during the procedure.

inlinestatic

Get the beam status.

This function determines the current beam status. The typical usage is to check if beam is switched on before (or during) an FIB exposure process.

Return values
Result
  • -1 unable to determine beam status,
  • 0 beam is off,
  • 1 beam is on,
  • 1000 on/off procedure is in progress
Timing
Executed immediately.
