Detailed Description

Energy filtering can be defined for multidetectors, inbeam f-BSE, .... This section describes:

  • list of all energy filters
  • enable/disable energy filtering
  • energy power settings
  • get current energy power
  • energy power limits

It is general, if energy filter is available for the detector, than the detector is present in enumration.

inlinestatic

Get list of detectors energy filters.

Map of detectors is returned.

The map has the following form:

det.0.name
det.0.id
det.0.unit
det.1.name
det.1.id
det.1.unit
...
det.N-1.name
det.N-1.id
det.N-1.unit
  • N is number of detectors with energy filtering.
  • name is human-readable detector name.
  • id is a numeric index
  • unit of energy filter

For future compatibility, application must silently ignore any key which does not have exactly the above form.

Returns
map containing detector energy filtering description
