Detailed Description

External SharkSEM application can indicate its progress in the Tescan Essence software. Arbitrary text and progress indicator overlay window is displayed in the Tescan Essence software. This feature can be useful, for example in case of long-taking spectrum acquisition or similar kind of applications.

There are no priorities among the remote sessions, there is only single progress dialog. "The last wins" strategy takes place.

Progress indication has been introduced in version 2.0.11.

Display progress dialog.

Shows progress dialog. If the dialog is already shown, its configuration is updated. Title and Text are zero-terminated messages encoded in UTF-8 strings.

HideButton controls whether the dialog contains a Hide button. If the Hide button is displayed, SEM user is allowed to hide the progress dialog manually.

If Marquee is set to one, dialog shows activity but does not indicate what proportion of the task is complete. If Marquee is set to zero, progress dialog shows specific percentage. The range is delimited by ProgressMin, i.e. 0 %, and ProgressMax, i.e. 100 %. The initial indicator position is equal to ProgressMin.

Example: Progress dialog range is set to ProgressMin = 50, ProgressMax = 150. Current position is set to 100 (see ProgressPerc()). Progress dialog now shows progress 50 %.
