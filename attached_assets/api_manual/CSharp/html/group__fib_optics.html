out string  Map 
)
inlinestatic

Get list of all available optical parameter.

The map has the following form:

param.0.name
param.0.count
param.0.unit
param.1.name
...
param.X.name
param.X.count
param.X.unit

The key is separated by two dots. The middle number is an index of the parameter.

Application should not expect that the indexes form a consecutive sequence. The index can be any integer number.

  • name human-readable name.
  • count 1 or 2 (one if only the first value is used, two otherwise)
  • unit %, mm, deg, or other physical unit

For future compatibility, application must silently ignore any key which does not have exactly the above form.

Returns
map containing optical parameters
Timing
Executed immediately.
