Detailed Description

GIS related functions.

Only selected microscopes are equipped with the GIS system, typically it is part of microscopes with FIB.

The documentation distinguishes between units and lines. Unit refers to a separate GIS assembly. For example, there can be single 5-GIS unit installed, which contains five independent lines. Another example is an SEM/FIB instrument equipped with three MonoGIS units, each containing single line.

The identifiers of lines are independent on units, that is, single integer identifier is sufficient to specify arbitrary line within whole SEM/FIB instrument.

Each line is equipped with cooling/heating circuit and a gas valve. The valve can be opened only when specific conditions are fulfilled (vacuum, temperature, ...)

All API functions behave in asynchronous manner. If the completion would take too long time, function returns immediately.

GIS error codes

Most API calls return following integer error codes:

Value Symbolic name Description
>0 Call-specific name Call-specific status
0 GISERR_OK Success
-1 GISERR_INACTIVE GIS inactive (eg. standby mode)
-2 GISERR_UNIT_NOT_INSTALLED GIS unit not installed
-3 GISERR_LINE_NOT_INSTALLED GIS line not installed
-4 GISERR_VACUUM Vacuum not ready
-5 GISERR_OUTGAS_RUNNING Outgas in progress
-6 GISERR_OUTGAS_REQUIRED Outgas is required
-7 GISERR_VALVE_OPEN Reservoir valve is already open
-8 GISERR_VALVE_CLOSED Reservoir valve is already closed
-9 GISERR_HEATING_ON Heating is already on
-10 GISERR_HEATING_OFF Heating is already off
-11 GISERR_TEMP_NOT_READY Heating/cooling off or temperature is not ready
-12 GISERR_BEAM_ON Not allowed while beam is on
-13 GISERR_NOT_CALIBRATED GIS stage is not calibrated
-14 GISERR_IN_MOTION GIS stage is in motion
-15 GISERR_POS_OCCUPIED The GIS target position is occupied by another device
-16 GISERR_ACTUATOR_ERROR Actuator failed
-17 GISERR_INVALID_PARAMETER Invalid command parameter(s)
-18 GISERR_PRECURSOR_BLOCKED GIS valve opening is blocked
-1000 GISERR_UNKNOWN Unknown error

Get list of available GIS lines.

Map of lines is returned.

Example result:

line.0.id=0
line.0.unit=0
line.0.name=Insulator
line.0.outgas_before_open=1
line.0.beam_off_before_close=0
line.0.beam_off_before_outgas=1
line.1.id=1
line.1.unit=0
line.1.name=Water
line.1.outgas_before_open=1
line.1.beam_off_before_close=0
line.1.beam_off_before_outgas=1
...
line.N.id=4
...

Where:

  • N + 1: the number GIS lines.
  • id: integer line id
  • unit: integer unit id
  • name: human readable name of the line
  • outgas_before_open: 1 - before opening the valve, outgas is required
  • beam_off_before_close: 1 - before closing the valve, beam must be switched off
  • beam_off_before_outgas: 1 - before outgas, beam must be switched off

For future compatibility, application must skip any unknown item.

Returns
list of GIS lines
Timing
Executed immediately.
