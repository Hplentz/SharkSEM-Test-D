Detailed Description

Each microscope is equipped with at least one image input channel. Each input is assigned one of the detectors. This section describes how detectors are controlled and assigned to input channels.

inlinestatic

Start automatic brightness/contrast adjustment procedure. Does not wait for procedure finish.

Channel index is a zero-based index of the input channel. Brightness and contrast of the active detector is adjusted to get optimal signal.

If the dwell parameter is omitted, current scanning speed is used instead. Refer to ScSetSpeed(). The dwell parameter is supported since protocol revision 3.0.0. It is recommended that all new client applications use the dwell parameter instead of the ScSetSpeed() call.

Current look up table and LUT source is used, refer to ScLUTParSet and ScLUTSrcSet. If no LUT is used, default values are used: min = 0, max = 100, gamma = 1.

If this function is called, appropriate detector must be selected as an active one first.

Uses histogram parameters of selected brightness/contrast strategy (if custom histogram parameters are not set)

Parameters
Channelinput video channel
Dwelldwell time [ns], optional
Timing
The execution time may vary. If waiting for D flag is not requested, it is executed immediately. If waiting D flag is requested, it will wait until procedure is done (approximately 1-10 seconds). Wait D flag is set during the movement procedure.

◆ DtEnable() [1/2]

static void SharkSEM.Commands.DtEnable ( this Connection  Conn,
int  Channel,
int  Enable 
)
inlinestatic

Enable/disable input channel.

This function is called before scanning is started. If input is enabled, image data is sent to the client. If input is disabled, image data is ignored.

If bpp differs from 8 or 16, the value is adjusted: 0 - 7 to 8; other values to 16.

Example:

DtSelect(0, 1); // select detector #1 into channel #0
DtEnable(0, 1); // enable channel #0, 8 bpp
DtAutoSignal(0); // adjust detector brightness/contrast
ScScanXY(...); // start scanning
// process image data
ScStop(...); // stop scanning

If 16 bits per pixel is selected, the ScData() callback has twice as much bytes in the data buffer than in 8-bit mode.

Parameters
Channelinput video channel index
Enable0 - disable, 1 - enable
Bppbits per pixel, optional 8 or 16, default is 8
Timing
Executed immediately.
See also
DtGetEnabled()
Call Context
Scanning must be inactive when calling this function.
Version
1.x.x
2.x.x
3.x.x
Protocol versions <= 1.0.5 do not have the bpp argument and use 8 bpp. Protocol versions >= 1.0.6 have optional bpp argument.

◆ DtEnumDetectors()

static void SharkSEM.Commands.DtEnumDetectors ( this Connection  Conn,
out string  DetectorMap 
)
inlinestatic

Get list of available detectors.

Map of detectors is returned.

The map has the following form:

det.0.name
det.0.detector
det.1.name
det.1.detector
...
det.N-1.name
det.N-1.detector
  • N is number of installed detectors.
  • name is human-readable detector name.
  • detector is a numeric detector index provided in Dt() functions, eg. DtSelect().

For future compatibility, application must silently ignore any key which does not have exactly the above form.

Returns
map containing detector description
Timing
Executed immediately.

Anytime
