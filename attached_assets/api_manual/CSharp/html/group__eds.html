Acquire EDS measurement.

Create a new EDS measurement identified by Handle and start acquisition. If any termination condition was specified, the acquisition stops automatically as soon as any termination condition is met. The acquisition can also be stopped prematurely by EdsStopAcq() command.

Parameters
Widthwidth of the whole scanning window
Heightheight of the whole scanning window
Leftx coordinate of topleft corner of observed area
Topy coordinate of topleft corner of observed area
Rightx coordinate of bottomright corner of observed area
Bottomy coordinate of bottomright corner of observed area
Profileid of a profile for this measurement (if profile id is not recognized, error is returned)
Countstermination condition, acquisition stops once desired counts have been collected. 0 - infinite acquisition unless Seconds is specified
Secondstermination condition, acquisition stops after seconds specified 0 - infinite acquisition unless Counts is specified
DwellTimepixel dwell time in [ns]
CoatingElementcoating element atomic number (0 - no coating element)
Return values
Handlemeasurement handle
Result0 - ok, handle is valid, < 0 - error, handle is invalid
Value Result Details
0 Ok
-1 Error Invalid Operation
-2 Error Beam Off
-3 Error SEM Scan start failed
-4 Error DPP Scan start failed
-50 Error Invalid profile
-99 Error Unknown
Timing
The execution time may vary. If waiting for A flag is not requested, it is executed immediately. If waiting A flag is requested, it will wait until procedure is done.
Call Context
EDS must be initialized and EDS state must be Idle (EdsGetState()).
See also
EdsGetState()

Get list of available EDS profiles.

Map of available profiles is returned

The map key has the following form:

profile.0.id=0
profile.0.name=Resolution
...
profile.N.id=1
...

Where

  • N + 1: the number of profiles
  • id: integer profile id
  • name: human readable name of the profile

For future compatibility, application must skip any unknown item.

Returns
list of EDS profiles
Timing
Executed immediately
Call context
Anytime

out int  Result

inlinestatic

Initialize the EDS environment.

The EDS environment needs to be initialized before any measurements are started.

A simple general workflow could look like this:

EdsInit(); // Initialize EDS
while (EdsState() != 0) { // Wait for EDS state Idle
Sleep(1000)
}
handle, result = EdsAcquire(...) // EDS Acquisition can now be started
Return values
Result1 - ok, 0 - error
See also
EdsGetState()
Timing
Executed immediately. The initialization runs in the background and can take several seconds.
Call Context
Call to this must occur before the first EDS Acquisition
Version
3.1.18

Quantify EDS measurement's composition.

The composition has the following form:

valid=1
fit=0.00
composition.0.element=14
composition.0.atomicPercent=50.00
composition.0.weightPercent=67.64
...
composition.N.element=28
...

Where

  • valid: quantity analysis validity (1 - valid, 0 - invalid)
  • fit: quantity analysis fit value
  • N + 1: the number of elements
  • element: element atomic number
  • atomicFraction: atomic fraction float value in [%] or NaN in case of unavailable value
  • weightFraction: weight fraction float value in [%] or NaN in case of unavailable value

For future compatibility, application must skip any unknown item.

Parameters
HandleEDS measurement handle
Elementslist of atomic numbers of elements to be included in quantity analysis
Returns
EDS measurement composition
Timing
Executed immediately, can take several seconds
Call context
Handle must identify an existing EDS measurement.
