Detailed Description

In the protocol versions 1.x.x and 2.x.x the FIB uses only secondary electron detector (SE). Wherever channel or detector field was required, its value was zero. For this reason, function equivalent to SEM DtSelect() was not present, because there are only single detector and single image channel.

Since protocol version 3.x.x FIB supports work with all available detectors - similar to SEM. According to this, new functions to work with detectors are added.

out string  DetectorMap

inlinestatic

Get list of available FIB detectors.

Map of detectors is returned. The map has the following form:

det.0.name
det.0.detector
det.1.name
det.1.detector
...
det.N-1.name
det.N-1.detector

Where:

  • N is number of installed detectors.
  • name is human-readable detector name.
  • detector is a numeric detector index provided in FibDt functions.

For future compatibility, application must silently ignore any key which does not have exactly the above form.

Returns
list of detectors
Timing
Executed immediately.
Call Context
Anytime
Version
3.x.x
