Detailed Description

FIB preset is a group of FIB parameters (aperture, centering, condenser excitation, working distance, etc.). The presets are defined by user in the FIB GUI. Each preset is identified by unique, user-defined name. SharkSEM can enumerate the presets and apply them.

In protocol version 2 FIB presets are separated from general presets. Since version 3.0.0 FIB presets are sub-set of all presets managed by Presets.

Get list of all available FIB presets.

The returned string contains list of preset names. The string is divided into lines, there is one preset name per line. If there are no presets, the string is empty.

Both system and user presets are listed. If both contain an identical name, user preset is preferred.

Update the values of a given FIB preset.

The user FIB presets are searched for the specified preset name. If the preset is found, the preset is updated and saved. Otherwise, the appropriate error is returned.

Note: This command can't be used to update system presets.

Parameters
Idname of the preset to be updated
Return values
Result0 - SUCCESS, 1 - USER_PRESET_NOT_FOUND, 2 - USER_PRESET_UPDATE_FAILED, 3 - USER_PRESET_VALIDATION_FAILED, 4 - FATAL_ERROR
Timing
Executed immediately.
Call Context
In order to get reproducible results, the microscope should be in a stable state.

Activate FIB preset.

The preset table is searched for the specified preset name. If it is found, the preset is activated. If the preset is not found, the request is ignored.

If given name exists in both user and system presets, user preset is activated.

Parameters
Idname of the preset to be activated
Timing
This call can take arbitrarily long time, depending on the previous state of the FIB subsystem. Before further processing, check FIB wait flags using IsBusy() call, or use conditional command execution.
Call Context
No FIB automatic procedure must be running, the FIB optics must be stable.
