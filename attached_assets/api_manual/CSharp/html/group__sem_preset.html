Detailed Description

SEM preset is a group of SEM parameters (aperture, centering, condenser excitation, working distance, etc.). The presets are defined by user in the SEM GUI. Each preset is identified by unique, user-defined name. SharkSEM can enumerate the presets and apply them.

SEM presets are sub-set of all presets described in section Presets.

SEM preset are new since version 3.0.0.

Get list of all available SEM presets.

The returned string contains list of preset names. The string is divided into lines, there is one preset name per line. If there are no presets, the string is empty.

Both system and user presets are listed. If both contain an identical name, user preset is preferred.

Returns
string containing list of SEM presets, one entry per line

Executed immediately.

Update the values of a given SEM preset.

The user SEM presets are searched for the specified preset name. If the preset is found, the preset is updated and saved. Otherwise, the appropriate error is returned.

Note: This command can't be used to update system SEM presets.

Parameters
Idname of the preset to be updated
Return values
Result0 - SUCCESS, 1 - USER_PRESET_NOT_FOUND, 2 - USER_PRESET_UPDATE_FAILED, 3 - USER_PRESET_VALIDATION_FAILED, 4 - FATAL_ERROR
Timing
Executed immediately.
Call Context
In order to get reproducible results, the microscope should be in a stable state.
Version
3.2.1 and later

Activate SEM preset.

The SEM preset table is searched for the specified preset name. If it is found, the preset is activated. If the preset is not found, the request is ignored.

If given name exists in both user and system presets, user preset is activated.

Parameters
Idname of the preset to be activated
Timing
This call can take arbitrarily long time, depending on the previous state of the SEM subsystem. Before further processing, check SEM wait flags using IsBusy() call, or use conditional command execution.
Call Context
No SEM automatic procedure must be running, the SEM optics must be stable.
