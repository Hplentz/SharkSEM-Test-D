Detailed Description

This chapter describes interface of RCA (Rotating Chord Algorithm) extension. This extension has been developed in cooperation with U.S. National Institute of Standards and Technology.

In G3 and G4, "SharkSEM RCA" license is required for operation.

For more information see Rotating Chord Algorithm.

Get effective scanning ramp DAC range.

The SEM scanning generator does not utilize the whole 16-bit DAC range (0 .. 65536), some space is reserved on both DAC range ends. This function tells what range is utilized by SEM scanning.

The value is same for all magnifications and scanning modes, it depends only on the device and software version.

Characterize next particle.

The whole particle is characterized automatically, client receives only callbacks. The callbacks contain all information about the particle.

The EDX analysis of current particle can be asynchronously stopped using RCASkipParticle() function.

The edx_mode specifies the particle scanning strategy. For more information see Modes.

Parameters
Single0 - all particles in the image frame 1 - single particle
EdxMode0 - no EDX characterization 1 - make EDX, dwell for specified time in the center 2 - make EDX, fixed resolution mesh 3 - make EDX, mesh with fixed spacing 4 - make EDX, dynamic re-centering mode
EdxParammode 0 - ignored, mode 1, 2, 3, 4 - time of EDX analysis [ms]
Call Context
RCA initialized and idle.

Specify set of callbacks sent to the client.

Following callback groups are defined:

  • 0x0001 frame start, frame finished events
  • 0x0002 line start, line done
  • 0x0004 RCA characterization events
  • 0x0008 send list of scanning coordinates
  • 0x0010 send list of image brightness values

By default, all callback are enabled.

Parameters
CbMaskcallback mask, binary 1 enables the callback

inlinestatic

Skip particle which is currently being characterized.

If particle is currently being EDX characterized (state 70, state 71), the remaining part of the EDX measurement is skipped. The RCA process then continues as if the particle was finished in a standard way.

If the specified p_ind particle is currently not being characterized, this function is ignored.
