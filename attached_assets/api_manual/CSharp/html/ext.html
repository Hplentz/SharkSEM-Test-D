Licensing

SharkSEM supports multiple license types - basic, advanced and automation. If there is no SharkSEM license at all, the server will not run and it is not possible to remotely connect and use the engine.

Basic license
Basic license is part of most Tescan microscopes and is needed to run the server itself and to use core functions.
Advanced license
Advanced license extends set of SharkSEM functions. This license is part of some most advanced engines or can be bought separately. If there isn't valid advanced license, the server discards all advanced function calls.
Automation license
Automation license brings a brand new scripting interface for an easier and more user-friendly microscope control. It also offers a set of new commands in the SharkSEM interface.
