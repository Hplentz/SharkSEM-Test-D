Detailed Description

There exist detectors, for example backscattered electron detector or cathodoluminescence detector, equipped with motorized travel. Such detectors can be inserted or retracted from a remote client application.

Refer also to the Nose space guard chapter, which describes how to check whether the shared space below the SEM objective is available.

Mechanical detector shutters, which prevent sensor damage, e.g. ion damage of EDS detector window, are controlled similar way as retractable detectors. Due to that, API described in this chamber can control motorized shutters as well.

As a general term, nose describes either detector nose (retractable part) or shutter nose.

The detector (shutter) indexes are zero based. As of 3.2.13, the index assignment is:

index detector
0 Standard BSE Detector #1
1 Standard CL Detector #1
2 Standard BSE Detector #2
3 Standard CL Detector #2
4 Standard EDS Shutter #1
5 Standard R-STEM Detector #1
6 Standard SEM Shutter #1
7 external TOF 2.0

Currently available noses can be enumerated using the 'CmdNoseEnum' function.

In the future, detectors or shutters can have up to three (X, Y, Z) actuators. Currently, only X axis is used.

Mechanically calibrate the detector (shutter) travel.

If the calibration is successful, NoseIsCalib() returns 1 when the procedure is finished.

The calibration is done in a safe manner, i.e. the end switch is located in the outside (retracted) position.

The current state of the calibration procedure can be polled using NoseIsBusy() function.

Parameters
Nosenose index
Returns
-1 invalid request (bad index, movement in progress, unable to start calibration)
0 calibration was started
Timing
The execution time may vary. If waiting for B flag is not requested, it is executed immediately. If waiting B flag is requested, it will wait until calibration is done. Wait B flag is set during the calibration procedure.
Call Context
Nose must be idle.

Read configuration of a nose.

Set of key=value pairs is returned.

Example return value:

Name=Standard BSE Detector #1
LimitMin=-45.0
LimitMax=0.0
WorkingPosition.0.Name=Working
WorkingPosition.0.Id=1
...
WorkingPosition.N.Name=WorkingN
WorkingPosition.N.Id=x

The limits are in [mm]. In the inserted position, the value is zero. In the retracted position, the value is negative. The nose name is a unique string.

This parameter set may be expanded in the future. Client application must handle unknown keys (ignore/skip them).

If the Nose is invalid, empty string is returned.

inlinestatic

Move nose to an arbitrary predefined position.

For nose_index 0 through 3, and 5,7 (detectors), there are following predefined positions:

Memory Detector Position Typical X value
0 Fully retracted -90.0
1 Inserted into working position 0.0
2 Not used Not moved
3 Safe position (retracted so that there is not mechanical conflict with other detectors) -50.0

More memory values for additional working positions may be available for any nose. @Use NoseEnum() to obtain all available working positions for each nose

For Nose 4 (shutter), in version 2.x.x there are following predefined positions:

Memory Detector Position Typical X value
0 Fully retracted (parking position) -90.0
1 Fully inserted (sensor covered) 0.0
2 Standby position (sensor not covered, partly retracted) -10.0
3 Safe position (sensor not covered, mechanically safe) -30.0

If there is potential mechanical conflict during nose manipulation, the movement request is discarded.

Parameters
Nosenose index
Memindex of a stored position
Returns
0 movement was started
-1 invalid request
-2 movement cancelled, collision detected
-3 movement start failed
-4 movement start failed, stage not calibrated
-5 movement start failed, stage not in suitable position
-6 movement start failed, sample holder not suitable
Timing
The execution time may vary. If waiting for B flag is not requested, it is executed immediately. If waiting B flag is requested, it will wait until movement is done. Time of movement depends on distance. Wait B flag is set during the movement procedure.
Call Context
Nose must be idle. Nose must be calibrated.
See also
