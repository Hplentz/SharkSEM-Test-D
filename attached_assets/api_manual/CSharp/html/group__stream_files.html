Detailed Description

TESCAN StreamFile (TSF) format is a text-based file format. The file consists of commands, allowing to generate arbitrary scanning patterns. This format uses .tsf extension. By using TESCAN StreamFile format it is easy to achieve simple automation when planning scanning procedures.

More about TESCAN StreamFiles.

TESCAN StreamFiles have been introduced in version 3.1.16.
