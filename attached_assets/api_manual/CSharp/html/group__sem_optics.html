Detailed Description

This set of functions controls how the electron optics is set up. The most important parameters are scanning mode, magnification, probe current and working distance.

inlinestatic

Start automatic column alignment procedure. Does not wait for procedure finish.

Depending on the scanning mode, OBJ Centering and IML Centering are adjusted.

Channel index is a zero-based index of the input channel. Appropriate detector must be selected as an active.

There are several important preconditions. If they are not fulfilled, the column centring may fail.

  • Good signal quality
  • Good structure on the specimen surface
  • Well focused image

If the dwell parameter is omitted, current scanning speed is used instead. Refer to ScSetSpeed(). The dwell parameter is supported since protocol revision 3.0.0. It is recommended that all new client applications use the dwell parameter instead of the ScSetSpeed() call.

Current look up table and LUT source is used, refer to ScLUTParSet and ScLUTSrcSet. If no LUT is used, default values are used: min = 0, max = 100, gamma = 1.

As a result, objective centring value is changed. Unfortunately, it is not possible to automatically verify the quality of the automatic alignment result.

Parameters
Channelinput video channel
Dwelldwell time in [ns], optional
Timing
The execution time may vary. If waiting for D flag is not requested, it is executed immediately. If waiting D flag is requested, it will wait until procedure is done (approximately 10 - 60 seconds). Wait D flag is set during the procedure.

static void SharkSEM.Commands.AutoWD ( this Connection  Conn,
int  Channel 
)
inlinestatic

Start automatic working distance (WD) adjustment procedure. Does not wait for procedure finish.

Channel index is a zero-based index of the input channel. Appropriate detector must be selected as an active first, the automatic procedure requires good signal quality and suitable specimen to achieve good result.

If MinWD and MaxWD are specified, the search range is restricted. In some cases, this improves the algorithm reliability.

If the dwell parameter is omitted, current scanning speed is used instead. Refer to ScSetSpeed(). The dwell parameter is supported since protocol revision 3.0.0. It is recommended that all new client applications use the dwell parameter instead of the ScSetSpeed() call.

To set dwell without limiting MinWD and MaxWD, call function with both MinWD and MaxWD equals 0.

Default LUT table is used: min = 0, max = 100, gamma = 1.

As a result, WD value is changed.

Parameters
Channelinput video channel
MinWDminimum allowed WD in [mm], optional, only with MaxWD
MaxWDmaximum allowed WD in [mm], optional, only with MinWD
Dwelldwell time in [ns]
Timing
The execution time may vary. If waiting for D flag is not requested, it is executed immediately. If waiting D flag is requested, it will wait until procedure is done (approximately 10 - 60 seconds). Wait D flag is set during the procedure.
See also
DtSelect()
Call Context
Scanning must be inactive.

Degauss the microscope column.

During the degauss procedure, all electromagnetic coils in the microscope column are fed with alternating current with decreasing amplitude. The aim is to minimize the magnetic remanence.

out string  PcMap 
)
inlinestatic

Get list of all PC indexes.

Deprecated:
Functions operating with PC are obsolete and will be removed, please use SetBeamCurrent() and GetBeamCurrent() instead.

The returned map has the following form:

pc.1.current
pc.2.current
...
pc.N.current

1 is the minimum index, N is the maximum index. The list of indexes is a continuous sequence and the corresponding values are increasing or decreasing as well. Currently there are 20 basic indexes (1 to 20), but this may change in the future.

current is the estimated probe current in pA (picoamps). Application should not expect that the value is accurate, because it depends on the acceleration voltage, gun centering, and other parameters.

For future compatibility, application must silently ignore any key which does not have exactly the above form

Note
For backward compatibility, remote control still uses the PC parameter. The usual PC range is 1 to 20. The higher is PC, the lower is the electron beam current.
Return values
PcMapmap, PC indexes, one per line
Timing
Executed immediately.

◆ GetOptParams()

static void SharkSEM.Commands.GetOptParams ( this Connection  Conn,
out string  Map 
)
inlinestatic

Get current optical parameters.

The map contains:

  • final.ImlLensRotation is traced rotation of the IML lens in [deg].
  • final.ImlAlignmentRotation is angle between IML alignment coils and scanning coils in [deg].
  • final.StigARotation is angle between stigmator A and scanning coils in [deg].
  • final.StigBRotation is angle between stigmator B and scanning coils in [deg].
  • final.ApertureAngle is aperture of the final lens in [deg]. The value is a half-angle of the electron beam cone.
  • final.SpotSize is approximate spot size in [nm]. See also GetSpotSize().
  • final.ScanPivotPosition is position of the scanning pivot point [mm]. The value is relative to the objective pole piece. The orientation is positive below objective pole piece, negative above objective pole piece. The orientation and units are same as for working distance. See also SMGetPivotPos().
  • final.ObjLensRotation is traced rotation of the conventional OBJ lens in [deg].
  • final.ObjLensFinalRotation is complete rotation produced by scanning coils and lenses below the scanning coils (objectives) in [deg].

Example contents:

final.ApertureAngle=1.237
final.SpotSize=261.040
final.ScanPivotPosition=-6.500
final.ObjLensRotation=0.000
